# 🐾 PetLux - Landing Page de Pet Shop Premium

Projeto desenvolvido para a disciplina de Desenvolvimento Web utilizando conceitos de Engenharia de Prompt e Inteligência Artificial.

## 📖 Sobre o Projeto

A PetLux é uma Landing Page moderna para um Pet Shop especializado em Banho e Tosa Premium.

O objetivo do projeto é apresentar os serviços da empresa de forma atrativa, moderna e responsiva, proporcionando uma excelente experiência ao usuário e incentivando o agendamento de serviços.

## ✨ Funcionalidades

- Layout moderno em Dark Mode
- Detalhes neon e animações CSS
- Design responsivo
- Seção de serviços com tempo estimado
- Destaque para serviço "Mais Vendido"
- Formulário de agendamento
- Validação de datas
- Máscara automática para telefone
- Navegação simples e intuitiva

## 🛠 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla JS)
